# SP-AC-BG - Българска счетоводна система

Счетоводна система за български фирми, съвместима с изискванията на еврозоната.

## Архитектура

```
sp-ac-app/
├── backend/          # Java Spring Boot + GraphQL
├── frontend/         # React + TypeScript + Tailwind
└── docs/             # Документация
```

## Технологии

### Backend
- Java 21
- Spring Boot 3.x
- Spring GraphQL
- PostgreSQL
- Flyway миграции

### Frontend
- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Apollo Client v4

## Бързо стартиране

### Backend
```bash
cd backend
./mvnw spring-boot:run
```
GraphQL Playground: http://localhost:8080/graphiql

### Frontend
```bash
cd frontend
npm install
npm run dev
```
UI: http://localhost:3000

## Модули

### Счетоводство
- Сметкоплан (йерархичен, до 4 нива)
- Журнални записи с дебит/кредит редове
- Поддръжка на валути и материални сметки
- Контрагенти (клиенти, доставчици, др.)

### ДДС
- ДДС операции (покупки/продажби)
- Ставки по ЗДДС
- ДДС декларации
- Експорт за НАП (DEKLAR.TXT, POKUPKI.TXT, PRODAJBI.TXT)

### Дълготрайни активи (ДМА)
- Регистър на активи
- Категории по ЗКПО (7 категории)
- Месечно изчисляване на амортизация
- Двойна амортизация (счетоводна + данъчна)
- Автоматично осчетоводяване

### Справки
- Оборотна ведомост
- Хронологичен регистър
- Главна книга
- Дневник на операциите
- Справки по контрагенти

### Банки
- Банкови профили
- Импорт на банкови извлечения
- Open Banking интеграция (Salt Edge)

### Други
- Управление на валути (ECB курсове)
- AI сканиране на документи
- Склад (материални сметки)
